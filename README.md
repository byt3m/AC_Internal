# AC_Internal

Simple internal hack for the game "Assault Cube".


Current features:
- Box ESP (only enemies).
- Name ESP (only enemies).
- Godmode.
- Infinite rifle ammo.
- Infinite pistol ammo.
- Infinite grenade ammo.
- Grenade "no reload / wait time".
